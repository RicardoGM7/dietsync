Este projeto foi desenvolvido como trabalho prático para a disciplina de Programação Funcional.
Trata-se de um sistema CRUD (Create, Read, Update, Delete) totalmente construído em JavaScript com uma regra de negócio específica: a aplicação do paradigma funcional.
O grande diferencial tecnológico é que todas as funções principais utilizam recursividade.
Laços de repetição tradicionais (como for ou while) foram deixados de lado para explorar a fundo as chamadas recursivas. 
Além do rigor técnico na lógica, o projeto conta com um front-end moderno, garantindo uma estética bonita e uma ótima experiência visual.

Tecnologias Usadas: JavaScript, Html5, Css3
